7-Layer Context Stack
L0
Language
检测用户输入语言，固定响应语言
zh en — injected as system directive
L1
Identity
"You are Vibe Pencil's AI assistant"
persona role definition
L2
Conversation History
完整对话历史，超 8k tokens 自动压缩
raw if <8k compressed if >8k
L3
Canvas State
当前画布的完整快照
YAML schema selected node
BuildSummary code context
L4
Task Definition
当前任务类型，决定 Agent 行为模式
discuss import implement refactor
L5
Skills
按 techStack 动态加载，懒加载策略
manifest only → chat
full content → build
L6
Constraints
Agent 操作权限边界
read-only → Canvas Agent
write enabled → Build Agent
L7
Output Format
控制响应格式，随 phase 切换
brainstorm → text-only
design/iterate → canvas-action

Brainstorm Skill Hard-Gate
HARD GATE brainstorm phase — canvas-action BLOCKED
在 brainstorm 阶段，即使 Skill 包含 canvas-action 指令，系统强制拦截，仅允许纯文本响应。防止 AI 在需求未确认时就修改画布。
canvas-action blocked node mutations blocked text response allowed discussion allowed unlocked at design phase
